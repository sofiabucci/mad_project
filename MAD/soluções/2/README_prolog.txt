First, an instance generated by partsRect.c must be put trough the .py converter.
Then it will be converted to a list of prolog facts.

Usage for the prolog program:

1 - swipl c_prolog.pl
2 - consult('yourinstance.pl'). %Must be on the same folder as the prolog program.
3 - output(Total).

The program will first return the total number of guards in an optimal solution,
then it will search for all optimal solutions with the same number of guards.
